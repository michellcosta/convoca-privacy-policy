Politica de Privacidade - Convoca

Ultima atualizacao: 15 de abril de 2026

O Convoca coleta e usa seus dados para fornecer o servico de gerenciamento de peladas.

1. Dados Coletados

  • Nome e e-mail (via login Google)
  • Foto de perfil
  • Dados de partidas e presencas inseridos por voce
  • Token de dispositivo para notificacoes push

2. Uso dos Dados

  • Gerenciar grupos, partidas e jogadores
  • Enviar notificacoes push relevantes
  • Sincronizar dados entre membros do grupo

3. Servicos de Terceiros

  • Supabase (banco de dados)
  • OneSignal (notificacoes push)
  • Google Sign-In (autenticacao)

4. Seguranca

Seus dados sao transmitidos via HTTPS e armazenados com criptografia.

5. Seus Direitos (LGPD)

Voce pode acessar, corrigir ou solicitar exclusao dos seus dados a qualquer momento.
